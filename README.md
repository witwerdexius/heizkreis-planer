# Heizkreis-Planer

Standalone Web-App für die Verlegeplanung von Fußbodenheizungs-Trockenbauplattensystemen.

## Features

- **Nordic Fos 161941** – 600×1200 mm EPS-Platten, Ø 16 mm Rohr, CC 150 mm
- **Mäander & Schnecke** – beide Fräsmuster wählbar
- **Raumzeichnung** – beliebiges Polygon per Klick/Tippen
- **Layout-Engine** – Sutherland-Hodgman Clipping, automatischer Versatzoptimierer (6×6 Grid)
- **Stückliste** – Voll-/Schnittplatten, Fläche, Verschnitt %
- **Grundriss-Import** – JPG / PNG / PDF hochladen, Maßstab kalibrieren
- **Mehrere Räume** pro Projekt
- **Mobil** – Touch-Events, Pinch-to-Zoom, Bottom-Tab-Bar

## Nutzung

Einfach `index.html` im Browser öffnen – keine Installation, kein Build-Schritt.
